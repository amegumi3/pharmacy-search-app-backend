# Preparation
このアプリでは、Yahoo!ジオコーダAPIを使用します。<br>
事前に[アプリケーションIDを用意](https://e.developer.yahoo.co.jp/dashboard/)し、
取得したIDを**API_KEY**に格納してご利用ください。
```
API_KEY = "取得したアプリケーションID"
```
